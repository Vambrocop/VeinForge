# VeinForge

面向 **大麦 / 小麦（单子叶平行脉）** 的叶脉性状自动定量工具。
经典图像处理底座 + 可插拔深度学习分割 +（第二阶段）热胁迫表型，
强调 **一键安装、批量处理、参数可复现**。

> 📌 **状态：设计 / 立项阶段（design phase）。** 暂无可运行代码；当前仅整理参考资料与设计方向。

## 愿景与范围（已确定的方向）

- **目标**：把现在 ImageJ 里手动描脉 / 手数自由末端 / 人工判脉级的叶脉定量流程，
  升级成自动、批量、可复现的工具。
- **MVP 图像类型**：透明化叶片的**平面脉网**显微图（不含横切面解剖 / 整叶照片）。
- **技术路线**：先用**经典 CV**（Frangi/Tubeness → 阈值 → 骨架化 → 图分析 → 距离变换），
  **零训练数据即可起步**；深度学习分割作为第二阶段可插拔模块。
- **工具形态**：`pip` 安装的 Python 包 + 批量 CLI + 轻量 GUI（倾向 napari，便于人工校正）。
- **数据存储**：结果用内置 **SQLite**；图像数据集**不入代码仓库**（建议单独数据仓库或 Zenodo）。

## 路线图

- **P1（MVP）** 经典 CV：总脉密度 / 脉宽 / 自由末端 / areole + 整叶形态 → CSV + SQLite + 质控叠加图。
- **P1.5** 单子叶**纵脉 / 横脉**分离、脉间距、束鞘排除。
- **P2** 深度学习分割（参考 LeafVeinCNN）+ **热胁迫表型**（在脉性状上做分类 / 回归）。

## 差异化（相对 GrasVIQ / phenoVein 等已有工具）

工程化 / 可复现 · 单子叶专用优化 · 深度学习鲁棒性 · **热胁迫表型（真正的新意）**。
