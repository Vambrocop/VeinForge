## 已验证的洞见(有数据支撑)
- **方法 > 模型大小**:clDice + 数据增强把叶脉 DL 的 IoU 从 0.48 → **0.66**,反超经典 0.52。对细脉,**连通性损失比堆大模型更值**。
- **经典 CV 别小看**:零训练就 0.52,小数据下能打败欠训的 DL。**先有可用基线,再上 DL**。
- **固定 holdout 是底线**:每轮换 val 集会让 IoU **跨轮不可比**(我们踩过:0.37 / 0.53 / 0.48 其实不能直接比)。
- **IoU 会"低估"真实质量**:预测常比**稀疏的人工真值**还完整 → 别只看 IoU,要看**叠加图**核对。
- **脉宽 = 2×EDT − 1**(半像素边界修正),几何上比 2×EDT 更准。
- **脉间距用"距离图对数众数"**(2023 *Appl. Plant Sci.*)比朴素均值稳。
- **负结果也要记**:视网膜→叶脉**迁移在当前小数据/小模型下无明显增益**(0.664 vs 0.663)。说明此规模"从零"已够;迁移要更大模型 / 更对口域(如其它单子叶)才可能体现价值。
- **数据搜寻结论(详尽,2026-06-15)**:公开仓库 Zenodo/Dryad/figshare/Kaggle/GitHub/HF **均无小麦/大麦叶脉+掩膜数据集**;最广的多支系集(*Macroevolutionary trends*, Zenodo 13300782,1000 种)也**仅 3 种禾草**(芦苇 Calamagrostis / 竹 Chusquea / 化石芦苇 Phragmites,**零谷类**)。→ **小麦/大麦 DL 必须靠自有图**,公共数据只够做通用预训练底座(且实测迁移增益微弱)。**别再花时间找公开谷类脉数据了。**

## 实用技巧 / 方法
- **"脉/血管恒为少数类"** → 占比 > 0.5 自动取反,稳健修标注极性 bug(STARE 就靠这条修的)。可推广到任何"前景应为少数"的掩膜清洗。
- **零人工标注引导**:经典 CV 自动描一版 → Ilastik 修几笔 → 当 DL 训练真值(免从头手画)。
- **合成 phantom 当"零数据验证骨架"**:不依赖真实数据就能验证整条管线,甚至**预训练 DL**(IoU 1.0)。
- **下→训→删循环**:大公共数据下载用完即删,模型(几 MB)留下 → 磁盘不涨、底座越攒越强。
- **切块(tiling)而非缩图**:保住细脉 + 多出样本,优于 128px 缩图。

## 值得探索的点子(未做)
- **视网膜血管 ≈ 叶脉**(拓扑相似)→ 迁移底座(STARE / DRIVE)。
- **自监督(SSL)** 在海量无标注叶图上学通用特征 → 叶专属底座。
- **主动学习**:让模型挑"最不确定"的图优先标,标注效率最大化。
- **超分辨**:低分图 → 恢复细脉。
- **clDice 不只当损失,也当 benchmark 指标**:量"拓扑/连通正确性",补 IoU 看不出的断脉。
- **胁迫表型 = 真正的发表新意**:多数工具只"量脉",少有把脉性状接到胁迫响应。
- **多维"比赛"观**:准确度只是一维;VeinForge 的位置 = **接近人工的准确度 + 碾压的速度/自动化/可复现/物种通用**。
- **免费云 GPU**(Colab / Kaggle)= 不买硬件就解开算力限制。

## 元方法(怎么把这工具做好)
- **杠杆排序**:数据 > 方法 > 分辨率 > 模型 > 调参 > 推理增强(**别本末倒置**先堆大模型)。
- **诚实第一**:宁可报一个**解释得通的低分**,不报一个好看但不可比 / 带 bug 的高分。
- **每个新模块都 TDD + 固定 holdout**;合成数据先跑通,再上真实数据。
